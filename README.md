# LitCommerce — Channel Listing UI

A self-contained, dependency-free HTML/CSS/JS mockup of the LitCommerce eBay
channel-listing screen and its **Add New Product** flows.

## What's inside
- **index.html** — the entire UI in a single file (no build step, no frameworks).

## Features
- eBay channel listing screen (search, filters, tabs, product table, pagination)
- **Add New Product** dropdown with four create paths:
  - Add Product Manually → full-form side drawer
  - Add Product with Image → AI modal
  - Add Product with URLs → AI modal
  - Add Product with Title → AI modal
- Drawer sections: Images, Item Information, Identifiers, Description,
  Required / Recommended / Optional / Custom Specifics, Compatibility,
  Pricing, Inventory, Variations, Item Disclosures (+ Add Document modal)

## Run it
Just open `index.html` in any browser — no installation required.

## Publish with GitHub Pages (optional)
1. Push this repo to GitHub.
2. Settings → Pages → Source: `main` branch, `/ (root)`.
3. Your UI will be live at `https://<username>.github.io/<repo-name>/`.
